# RTMPCHybridEngine-WIN
基于RTMP和RTC混合引擎的在线视频连麦互动直播

Windows原生直播（网络自适应码率RTMP publisher）、点播播放器（播放器经过专业优化，可实现秒开RTMP Player）、基于RTMP和RTC混合引擎的视频连麦互动（最多支持4人同时互动）、高级版本支持直播控制台、插播广告
